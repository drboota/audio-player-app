# مشغل الصوت - تطبيق أندرويد

تطبيق مشغل صوت أندرويد يتيح لك تشغيل ملفات الصوت بجميع الامتدادات، وتسجيل الصوت، وتصفح ملفاتك.

## المميزات

- تشغيل ملفات الصوت بجميع الامتدادات الشائعة (MP3, WAV, OGG, M4A, FLAC, AAC)
- تسجيل الصوت بجودة عالية
- تصفح ملفات الصوت على جهازك
- تحكم كامل في التشغيل (تشغيل، إيقاف، إيقاف مؤقت، التالي، السابق)
- زر إعادة التشغيل للأبد (تكرار مستمر)
- واجهة عربية سهلة الاستخدام

## خطوات إنشاء ملف APK

### الخطوة 1: إنشاء مستودع على GitHub

1. اذهب إلى [GitHub](https://github.com)
2. سجّل الدخول أو أنشئ حساب جديد
3. اضغط على **New repository** (مستودع جديد)
4. أدخل اسم المستودع: `audio-player-app`
5. اختر **Public** (عام)
6. اضغط **Create repository**

### الخطوة 2: رفع الملفات

بعد إنشاء المستودع، افتح Terminal وأكتب:

```bash
# انتقل إلى مجلد المشروع
cd audio-player-app

# أضف المستودع البعيد
git remote add origin https://github.com/USERNAME/audio-player-app.git

# استبدل USERNAME باسم حسابك على GitHub
```

### الخطوة 3: رفع الكود

```bash
# أضف جميع الملفات
git add .

# اكتب رسالة commit
git commit -m "إضافة تطبيق مشغل الصوت"

# ارفع الملفات
git push -u origin main
```

### الخطوة 4: انتظر بناء APK

بعد رفع الملفات، سيبدأ GitHub Actions تلقائياً ببناء ملف APK:

1. اذهب إلى مستودعك على GitHub
2. اضغط على **Actions** (إجراءات)
3. سترى عملية البناء جارية
4. انتظر حتى تنتهي (عادة 5-10 دقائق)
5. اضغط على اسم العملية
6. اضغط على **audio-player-apk** لتحميل ملف APK

## تحميل ملف APK

بعد اكتمال البناء، يمكنك تحميل ملف APK من:

- **Actions** > **Build Android APK** > **audio-player-apk** > **app-debug.apk**

## تثبيت التطبيق

1. حمّل ملف `app-debug.apk` على هاتفك
2. افتح الملف على هاتفك
3. قد تحتاج تفعيل "المصادر غير المعروفة" في إعدادات الأمان
4. ثبّت التطبيق

## متطلبات التشغيل

- أندرويد 5.0 أو أعلى
- صلاحيات الوصول للملفات والتسجيل

## هيكل المشروع

```
audio-player-app/
├── src/
│   ├── App.tsx          # التطبيق الرئيسي
│   ├── main.tsx         # نقطة الدخول
│   └── index.css        # الأنماط
├── android/             # مشروع أندرويد
├── dist/                # ملفات البناء
├── package.json         # الحزم
├── capacitor.config.ts  # إعدادات Capacitor
└── README.md           # هذا الملف
```

## التطوير محلياً

```bash
# تثبيت الحزم
npm install

# تشغيل في المتصفح
npm run dev

# بناء التطبيق
npm run build

# إضافة منصة أندرويد
npx cap add android

# مزامنة مع أندرويد
npx cap sync android

# فتح في Android Studio
npx cap open android
```
