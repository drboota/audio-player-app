# مشغل الصوت - مواصفات التطبيق

## 1. الرؤية والمفهوم

تطبيق مشغل صوت احترافي يوفر تجربة سلسة ومتكاملة لتشغيل ملفات الصوت بأنواعها المختلفة، مع إمكانية التسجيل بأ جودة عالية وتصفح ملفات الجهاز بسهولة. التصميم يركز على البساطة والسهولة في الاستخدام مع واجهة عربية أنيقة.

## 2. لغة التصميم

### الاتجاه الجمالي
- تصميم عصري مسطح مع تدرجات لونية هادئة
- ألوان داكنة مريحة للعين مع لمسات نيون

### لوحة الألوان
- **الأساسي**: `#6C63FF` (بنفسجي)
- **الثانوي**: `#00D9FF` (سماوي)
- **الخلفية**: `#0F0F1A` (أسود داكن)
- **سطح**: `#1A1A2E` (رمادي داكن)
- **النص**: `#FFFFFF` (أبيض)
- **النص الثانوي**: `#8B8B9A` (رمادي)

### الخطوط
- **العناوين**: Cairo Bold
- **النص العادي**: Cairo Regular

### نظام المسافات
- وحدة أساسية: 8px
- مسافات: 8, 16, 24, 32, 48

### فلسفة الحركة
- انتقالات سلسة: 200-300ms ease-out
- تأثيرات نبض للأزرار النشطة
- شريط التقدم متدفق

## 3. الهيكل والتخطيط

### المكونات الرئيسية
1. **شريط التنقل السفلي**: تبويبات المشغل/التسجيل/الملفات
2. **صفحة المشغل**: تحكم كامل مع أزرار التشغيل
3. **صفحة التسجيل**: أزرار التسجيل مع مؤقت
4. **صفحة الملفات**: مستعرض ملفات بمجلدات

## 4. الوظائف والتفاعلات

### مشغل الصوت
- تشغيل/إيقاف
- التالي/السابق
- شريط تقدم تفاعلي
- التحكم في الصوت
- إعادة للنهاية (تكرار الملف كاملاً)
- دعم جميع الامتدادات: MP3, WAV, AAC, FLAC, OGG, M4A

### التسجيل
- زر تسجيل كبير
- عرض الوقت المسجل
- اختيار امتداد التسجيل
- قائمة التسجيلات

### تصفح الملفات
- عرض الملفات الصوتية
- تشغيل مباشر من الملف
- البحث في الملفات

## 5. المتطلبات التقنية

- Capacitor للإطار
- React مع TypeScript
- Web Audio API للتشغيل
- MediaRecorder للتسجيل
- File System Access API للتصفح
